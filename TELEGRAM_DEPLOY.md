# Запуск сайта как Telegram Mini App

## 1. Захостить сайт на HTTPS

Telegram **обязательно** требует HTTPS. Самый простой бесплатный вариант — **GitHub Pages**.

### Способ А — GitHub Pages (рекомендую)

```bash
cd ~/vedmy-club
git init
git add .
git commit -m "vedmy club mini app"
# Создайте репозиторий на github.com, потом:
git remote add origin https://github.com/ВАШ_ЛОГИН/vedmy-club.git
git branch -M main
git push -u origin main
```

Затем в репозитории на GitHub:
1. Settings → Pages
2. Source: `Deploy from a branch` → Branch: `main` → folder `/ (root)`
3. Через 1-2 минуты сайт доступен по адресу:
   `https://ВАШ_ЛОГИН.github.io/vedmy-club/`

### Способ Б — Vercel / Netlify

Перетащите папку `vedmy-club` на https://vercel.com/new или https://app.netlify.com/drop — получите HTTPS-адрес за 30 секунд.

## 2. Создать Telegram-бота

1. Откройте в Telegram → `@BotFather`
2. Команда `/newbot`
3. Введите имя бота (например, «Ведьмы не стареют»)
4. Введите username (должен заканчиваться на `bot`), например `vedmy_club_bot`
5. Получите **токен** — сохраните его

## 3. Привязать Mini App к боту

В `@BotFather`:

1. `/newapp` → выберите вашего бота
2. Введите название приложения: `Ведьмы не стареют`
3. Короткое описание: `Закрытый женский клуб`
4. Загрузите иконку 640x360 px (можно скриншот сайта)
5. **Web App URL**: вставьте URL вашего сайта из шага 1
6. Введите короткое имя приложения (slug, например `app`)

Готово! Открыть Mini App можно по ссылке:
`https://t.me/vedmy_club_bot/app`

## 4. Сделать кнопку «Открыть приложение» в боте

В `@BotFather`:

1. `/mybots` → выберите бота → `Bot Settings` → `Menu Button`
2. Введите название кнопки (например, «Открыть клуб»)
3. Введите URL вашего сайта

Теперь в любом чате с ботом снизу будет синяя кнопка, открывающая Mini App.

## 5. Тест

1. Откройте бота в Telegram (мобильный или desktop)
2. Нажмите кнопку «Открыть клуб» или перейдите по ссылке `t.me/ВАШ_БОТ/app`
3. Сайт откроется внутри Telegram

## Что уже работает в Mini App

- ✅ Авто-разворачивание на весь экран
- ✅ Имя пользователя автоматически подставляется в анкету
- ✅ Тактильная отдача (вибрация) на нажатия кнопок
- ✅ Telegram MainButton «Отправить анкету» появляется, когда форма видна
- ✅ Системная шапка перекрашена в цвет клуба
- ✅ Поддержка safe-area для iPhone

## Что НЕ работает (нужен бэкенд)

- ❌ Заявки хранятся только в браузере пользователя (localStorage).
  Чтобы все заявки приходили вам, нужен сервер. Самый простой путь:
  отправлять заявку через бота (метод `sendData` или `answerWebAppQuery`).

Если нужно — могу сделать бэкенд на Node.js/Python с базой и интеграцией с ботом.
